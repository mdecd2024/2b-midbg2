 # 網際綜合球類競賽機器人協同設計與模擬
 球場參考規格:

競技球: 白色, 直徑 0.1m, 重量 0.5kg

場地: 長 4m x 寬 2.5m, 外圍以足夠高度之隱形柵欄區隔

足球門: 長 0.6m, 高 0.3m, 寬 0.1m

籃板: 長 0.3m, 高 0.3m, 寬 0.1m, 位於足球門上方中央

籃框: 由籃板向場內距離 0.1m, 內徑 0.2m, 框管直徑 0.02

球員尺寸範圍: 長寬高各 0.2m, 重量 5kg
