cmsimde

SMap - SiteMap - 依照階次列出網站的所有頁面.

EditA - Edit All page - 將所有頁面放入編輯模式中, 主要用來處理頁面搬遷, 刪除或決定衝突頁面內容版本.

Edit - Edit page - 先選擇要編輯的單一頁面後, 再點選 Edit, 即可進入單一頁面的編輯模式.

Config - Configure Site - 編輯頁面標題與管理者密碼.

Search - 動態頁面內容的關鍵字搜尋.

IUpload - Image file Upload - 圖檔的上傳功能, 可以上傳 jpg, png 與 gif 檔案, 其中若在手機上傳圖檔, 系統會自動將圖片檔案縮小為 800x800 大小.

IList - Image file List - 列出可以直接在頁面編輯模式中引用的圖片檔案.

FUpload - File Upload - 一般檔案的上傳功能, 目前可以上傳的檔案副檔名包括 'jpg', 'png', 'gif', '7z', 'pdf', 'zip', 'ttt', 'stl', 'txt', 'html', 'mp4' 等, 使用者可以自行修改.

FList - File List - 列出可以直接在頁面編輯模式中引用的上傳檔案.

Logout - 登出頁面編輯模式.

Convert - 將動態網站中位於 config/content.htm 檔案, 透過分頁設定轉為 content 目錄中的靜態網頁.

acp - git add, git commit 與 git push, 通常只有在 localhost 或自架主機上利用網頁表單協助將倉儲改版內容推向 Github 倉儲.

SStatic - Start Static Site - 利用 Python 啟動網站伺服功能, 可以讓使用者檢查轉檔後的靜態網站內容.

RStatic - Replit Static Site URL - 僅用於 Replit 模式, 當使用者按下 SStatic 後, 可以按下 RStatic 進入當下尚未推向 Github Pages 的靜態網站.

80 - 由 init.py 中 static_port 所決定的連結字串, 一般不使用 80, 只有在 Replit 中為了與動態網站共用 port, 才特別設為 80.

文章連結

LaTeX

\documentclass{article}
\usepackage[utf8]{inputenc}

% 定义英文到中文的映射
\def\englishToChinese{
{"hello", "你好"},
{"world", "世界"},
{"programming", "编程"},
{"language", "语言"},
% 添加更多映射
}

% 定义翻译函数
\newcommand{\translateToChinese}[1]{
\foreach \word/\translation in \englishToChinese {
\ifx\word#1
\translation
\break
\fi
}
#1
}

\begin{document}

% 输入英文句子
\def\englishSentence{hello world, programming language.}
\textbf{English sentence:} \englishSentence

% 翻译并输出结果
\textbf{Translated sentence:}
\translateToChinese{hello}
\translateToChinese{world},
\translateToChinese{programming}
\translateToChinese{language}.

\end{document}

c++翻譯程式

#include <iostream>
#include <map>
#include <string>
#include <sstream>
#include <vector>

using namespace std;

// 定義英文到中文的映射
map<string, string> englishToChinese = {
{"hello", "你好"},
{"world", "世界"},
{"programming", "編程"},
{"language", "語言"},
// 添加更多映射
};

// 函數: 翻譯英文句子為中文
string translateToChinese(const string& sentence) {
stringstream ss(sentence);
string word;
vector<string> translatedWords;

while (ss >> word) {
// 尋找該單詞在映射中的翻譯，如果找不到，則保留原單詞
auto it = englishToChinese.find(word);
if (it != englishToChinese.end()) {
translatedWords.push_back(it->second);
} else {
translatedWords.push_back(word);
}
}

// 將翻譯後的單詞組合成句子
string translatedSentence;
for (const string& word : translatedWords) {
translatedSentence += word + " ";
}

return translatedSentence;
}

int main() {
// 輸入英文句子
string englishSentence;
cout << "Enter an English sentence: ";
getline(cin, englishSentence);

// 翻譯並輸出結果
string translatedSentence = translateToChinese(englishSentence);
cout << "Translated sentence: " << translatedSentence << endl;

return 0;
}
